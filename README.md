### git bash 계정 변경
1. 로그인된 계정/이메일 확인
  + git config user.name
  + git config user.email

2. 두번째



### 로그인 계정/이메일 변경
. git config --global user.name <변경할 이름>
. git config --global user.email <변경할 이메일>
